# DOM-Surgeon 🛠️
> **Opérez le web. Ne gardez que l’essentiel.**

DOM-Surgeon est une extension de navigateur puissante et intuitive qui vous redonne le contrôle de votre expérience de navigation. Marre des distractions, des bannières intrusives ou des sections inutiles ? Devenez le chirurgien de votre propre interface web.

## ✨ Fonctionnalités
* **Ciblage Chirurgical** : Survoler n'importe quel élément HTML pour le mettre en évidence avec une précision visuelle.
* **Ablation Instantanée** : Un simple clic permet de masquer l'élément sélectionné et d'épurer la page.
* **Interface Moderne** : Un panneau de contrôle sombre et élégant intégré directement à votre page.
* **Réinitialisation Facile** : Annulez toutes vos modifications d'un seul clic pour retrouver la page originale.

## 🚀 Installation (Mode Développeur)
Comme cette extension n'est pas encore sur le Chrome Web Store, voici comment l'installer gratuitement :

1.  Téléchargez ce dépôt sous format `.zip` et extrayez-le.
2.  Ouvrez votre navigateur (Chrome, Edge ou Brave) et accédez à `chrome://extensions/`.
3.  Activez le **Mode développeur** (en haut à droite).
4.  Cliquez sur **Charger l'extension décompressée** et sélectionnez le dossier contenant les fichiers du projet.

## 🛠️ Stack Technique
* **Manifest V3** : Conforme aux dernières normes de sécurité de Google Chrome.
* **JavaScript (Vanille)** : Injection dynamique de scripts et manipulation du DOM en temps réel.
* **CSS3** : Styles personnalisés pour une interface utilisateur fluide et réactive.
* **HTML5** : Structure du popup d'accueil.

## 📂 Structure du Projet
* `manifest.json` : Configuration et permissions de l'extension.
* `surgeon.js` : Le cœur de la logique de manipulation du DOM.
* `popup.html` : L'interface utilisateur de la barre d'outils.
